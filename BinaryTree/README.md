为什么要求要用完全二叉树？

因为完全二叉树，我们在给节点编序号的时候是连续的，并且序号是从1开始而不是从0开始的。

![image-20211102095952201](README.assets/image-20211102095952201.png)

为什么二叉树序号从1开始？

这是因为子节点除以2，再向下取整等于父节点的序号
